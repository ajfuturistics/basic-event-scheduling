## ENV

```bash
BASE_URL=http://localhost:3000
NEXT_PUBLIC_SERVER_URL=http://localhost:3000
```

## Steps to run

First, run the development server:

```bash
git clone https://github.com/ajfuturistics/basic-event-scheduling.git

cd basic-event-scheduling

npm install

npm run dev
```
